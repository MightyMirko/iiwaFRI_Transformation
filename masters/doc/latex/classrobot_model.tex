\doxysubsection{Detailed Description}
Class representing the robot model and providing kinematic information. 

The \mbox{\hyperlink{classrobot_model}{robot\+Model}} class is responsible for managing the robot model, performing forward kinematics, and providing information about the robot\textquotesingle{}s transformation.
