\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_k_u_k_a_1_1_f_r_i}{FRI}} client interface. 

This is the callback interface that should be implemented by \mbox{\hyperlink{namespace_k_u_k_a_1_1_f_r_i}{FRI}} clients. Callbacks are automatically called by the client application (\mbox{\hyperlink{class_k_u_k_a_1_1_f_r_i_1_1_client_application}{Client\+Application}}) whenever new \mbox{\hyperlink{namespace_k_u_k_a_1_1_f_r_i}{FRI}} messages arrive.
