\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_k_u_k_a_1_1_f_r_i}{FRI}} client application class. 

A client application takes an instance of the \mbox{\hyperlink{class_k_u_k_a_1_1_f_r_i_1_1_i_connection}{IConnection}} interface and an instance of an \mbox{\hyperlink{class_k_u_k_a_1_1_f_r_i_1_1_i_client}{IClient}} interface to provide the functionality needed to set up an \mbox{\hyperlink{namespace_k_u_k_a_1_1_f_r_i}{FRI}} client application. It can be used to easily integrate the \mbox{\hyperlink{namespace_k_u_k_a_1_1_f_r_i}{FRI}} client code within other applications. The algorithmic functionality of an \mbox{\hyperlink{namespace_k_u_k_a_1_1_f_r_i}{FRI}} client application is implemented using the \mbox{\hyperlink{class_k_u_k_a_1_1_f_r_i_1_1_i_client}{IClient}} interface.
